🚀 QUICK START
════════════════════════════════════════════════════════════════════════════

Option 1: Using Setup Script (Recommended)
  
  Linux/macOS:
    chmod +x setup.sh
    ./setup.sh
  
  Windows:
    setup.bat

Option 2: Using Docker

  docker-compose up
  Open: http://localhost:3000

Option 3: Manual Setup

  Terminal 1:
    cd server
    npm install
    cp .env.example .env
    npm run dev

  Terminal 2:
    cd client
    python -m http.server 3000

  Terminal 3:
    mongod

  Browser:
    http://localhost:3000
